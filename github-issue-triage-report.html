Project Goal Summary

The project aims to build a small, auditable Flue-based autoresearch harness for deciding when skill context is useful, comparing no-skill and skill-guided performance, improving the smallest useful skill, and preserving enough artifacts to reconstruct each run. The current roadmap is a dependable single-skill loop: users can install the harness, run or import a baseline, complete or resume an iteration, understand the evidence, and rerun safely without hand-editing Flue payloads or deleting artifacts manually. Reliability, auditability, documentation, and operator usability therefore outrank broader post-alpha capabilities.
