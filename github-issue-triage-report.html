Project Goal Summary

The project aims to build a small, auditable Flue-based autoresearch harness for deciding when skill context is useful, comparing no-skill and skill-guided performance, improving the smallest useful skill, and preserving enough artifacts to reconstruct each run. Near-term work should tighten reliability, auditability, cost visibility, documentation, schema-validated outputs, and recovery around the single-skill release-notes alpha loop before broadening the surface area.
